## Key Insights (2026-01-27)

**Selene's unique position:**
- ADHD-focused + local LLM (others are one or the other)
- Librarian/archivist role, not productivity app
- Capture-first design (Drafts integration)
- Background processing via launchd (not Electron)
- Native macOS app (Swift/SwiftUI)

**The librarian metaphor:**
- Captures without judgment
- Maps connections automatically
- Researches on demand
- Surfaces proactively
- Never loses anything

**Spatial vision (future):**
- Cork board / infinite canvas
- 3D cube storage (x/y position + z layers)
- Auto-clustering with manual override
- Freeform-style zoom navigation
- Chat-driven placement
